# Streamlytics - Movie Streaming Analytics (MySql)
An analytical SQL-based project designed to explore user behavior, content performance, and subscription-driven revenue insights for a movie streaming platform, similar to real-world OTT services like Netflix or Prime Video.

## Pupose:
Streamlytics is a MySQL analytics project that transforms raw streaming platform data into actionable insights.
The project focuses on understanding what users watch, how they engage with content, and how subscriptions contribute to revenue, enabling data-driven decisions in content strategy, marketing, and pricing.
